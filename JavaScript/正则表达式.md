# 正则表达式

## 基本规则及含义
```javascript
/……/              // 中间写正则表达式

^……               // 从什么开始匹配

……$               // 以什么作为匹配的结尾

+                 // 匹配一次或多次

*                 // 匹配任意次数（零次或一次或多次）

?                 // 匹配零次或一次

[a-z]             // 匹配 a 到 z 之间的任意小写字母

[A-Z]             // 匹配 A 到 Z 之间的任意大写字母

[0-9]             // 匹配 0 到 9 之间的任意数字

……{n}             // 将什么进行 n 次匹配（n 是一个非负整数）

……{n, }           // 将什么进行至少 n 次匹配（n 是一个非负整数）

……{n, m}          // 将什么进行至少 n 次匹配，且至多进行 m 次匹配（m 和 n 均为非负整数，其中n <= m）
```

## 常用匹配规则

### 匹配手机号码
```javascript
/^1[5|8][0-9]{9}$/  // 以 1 开头，第二位是固定的数字，第三位开始到第十一位就是随机的数字组成了
```

### 匹配邮箱地址
```javascript
/^[1-9a-zA-Z_]+@{1}[1-9a-zA-Z]+\.com$/  // 以数字字母和下划线混排开始，接着是 @ 符号，之后是企业域名，数字和字母的混排，最后以 .com 结尾
```
